Idle Companion

v2.7.1 · Chrome / Edge / Brave 扩展

这是什么

当你在浏览器里长时间无操作（无鼠标、键盘、滚动等），屏幕上会出现 透明抠图的写实动物， 像 Cat Gatekeeper 一样挡在页面最前面 —— 背后网页仍可见。 有任何操作或停留时间结束后自动消失。

功能

🐱 猫咪Pexels 实拍舔毛循环，全屏透明 WebM
🐋 蓝鲸透明入场 + 待机循环
🦊 狐狸透明入场 + 待机循环
⏱ 可调计时空闲多久出现 / 停留多久
📋 白名单仅在指定网站生效
🔊 音效Web Audio 合成，可关闭

安装（开发者模式）

  1. 克隆或下载本仓库，找到 idle-companion/ 目录
  2. 打开 chrome://extensions（Edge 为 edge://extensions）
  3. 开启右上角 开发者模式
  4. 点击 加载已解压的扩展程序，选择 idle-companion 文件夹
  5. 更新代码后点扩展卡片上的 重新加载，并刷新测试页
git clone https://github.com/shuaianchen-glitch/snake-game.git
→ 选择 snake-game/idle-companion 文件夹加载

快速体验

  1. 点击工具栏图标，选角色 猫咪
  2. 将「空闲多久后出现」设为 1 分钟
  3. 打开任意网页，停止操作，等待倒计时结束

# Idle Companion — 空闲陪伴浏览器插件

当你在浏览器中长时间无操作（无鼠标、键盘、滚动等），屏幕上会出现拟真动画角色陪伴你。

## 功能

- **空闲检测**：监测鼠标、键盘、滚动、触摸等操作，无操作时开始计时
- **倒计时显示**：右下角显示距离角色出现的剩余时间
- **自定义时间**：
  - 空闲多久后出现（默认 5 分钟，范围 1–120 分钟）
  - 出现后停留多久（默认 3 分钟，范围 1–60 分钟）
- **透明抠像叠加（蓝鲸）**：实时去除海水蓝色背景，动物直接出现在你正在浏览的网页上方
- **全屏写实视频（狐狸/猫咪）**：完整视频铺满屏幕（暂未抠图）
- **出场音效**：Web Audio 合成音效，每种角色有独特的出场与动作声音（可关闭）
- **网站白名单**：可设置仅在指定网站生效，支持 `github.com` 或 `*.google.com` 格式
- **自动消失**：停留时间结束后淡出；用户有任何操作也会立即消失
- **刷新消失**：刷新或离开页面时，动画自动清除

## 安装（Chrome / Edge / Brave）

1. 打开浏览器扩展管理页：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本目录：`idle-companion/`
5. 若已安装旧版本，点击扩展卡片上的 **重新加载** 按钮（v1.3.0 起为视频版，必须重新加载）

## 使用

1. 点击浏览器工具栏中的插件图标打开设置面板
2. 选择角色，调整时间与音效
3. （可选）开启白名单并添加网站
4. 保持页面打开且不做任何操作，等待倒计时结束

### 白名单

- 默认关闭：所有网站均生效
- 开启后：仅列表中的域名会触发陪伴动画
- 支持格式：
  - `github.com` — 匹配 github.com 及其子域名
  - `*.google.com` — 仅匹配 google.com 的子域名
- 点击 **添加当前网站** 可快速添加你正在浏览的页面

### 快速测试

1. 将「空闲多久后出现」改为 **1** 分钟
2. 若启用了白名单，确保当前网站已添加
3. 停止操作页面，约 1 分钟后即可看到效果

## 项目结构

```

├── assets/                # 写实视频素材（可替换）
│   ├── whale.mp4
│   ├── fox.mp4
│   └── cat.mp4
├── content/
│   ├── content.js         # 空闲检测、白名单、角色调度
│   ├── sounds.js          # Web Audio 合成音效
│   ├── styles.css
│   └── characters/
│       ├── base.js
│       ├── whale.js
│       ├── fox.js
│       └── cat.js
├── popup/                 # 设置面板
└── icons/
```

### 透明抠像模式（蓝鲸）

选择 **蓝鲸** 角色时，使用 **AI 预处理的透明 WebM 视频**（非实时抠像）：
- 背景完全透明，浏览器页面清晰可见
- 只有海豚/海洋动物叠加在页面上
- 边缘干净，无马赛克

狐狸和猫咪目前仍是完整视频（带背景），后续可用同样技术扩展。

## 替换视频素材

将你自己的 MP4 视频放入 `assets/` 目录，保持文件名不变即可：
- `whale.mp4` — 蓝鲸/海洋动物
- `fox.mp4` — 狐狸
- `cat.mp4` — 猫咪

建议：横屏、720p 以上、10–30 秒、动物占画面主体。

当前内置素材来自 [Mixkit](https://mixkit.co) 免费视频库。

## 扩展更多角色

在 `content/characters/` 下新建角色类，继承 `CharacterBase`，然后在 `content.js` 的 `CHARACTERS` 对象和 `popup.html` 中注册。如需专属音效，在 `sounds.js` 中添加对应的 `playEntrance` / `playAction` 方法。

## 技术说明

- Manifest V3，权限：`storage`（保存设置）、`activeTab`（获取当前标签页域名）
- 音效通过 Web Audio API 实时合成，无需加载外部音频文件
- 白名单通过 `location.hostname` 匹配，内容脚本在所有网页注入但在非白名单站点不激活
